# Interactive Resume Builder

An interactive Resume Builder web application created using HTML, CSS, and Vanilla JavaScript.

This project allows users to create a professional resume dynamically through a step-by-step form interface with real-time preview functionality.

---

## Features

- Step-by-step resume form
- Live resume preview
- Edit functionality for every section
- Dynamic skills list generation
- Clean and responsive UI
- Final resume save mode
- Beginner-friendly JavaScript project

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## Sections Included

- Personal Details
- Education
- Experience
- Skills
- Projects

---

## How It Works

1. Fill personal information
2. Add education details
3. Enter experience details
4. Add skills
5. Add projects
6. Save and preview final resume
